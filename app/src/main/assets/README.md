# Инструкция по добавлению SVG файлов

## Размещение SVG файла

1. Поместите ваш SVG файл карты 2 этажа в эту папку (`app/src/main/assets/`)
2. Назовите файл `floor2.svg` (или измените название в `MainActivity.kt`)

## Настройка интерактивных областей

После добавления SVG файла вам нужно настроить координаты кабинетов в `MainActivity.kt` в методе `setupRoomAreas()`.

### Как найти координаты кабинетов:

1. Откройте SVG файл в редакторе (Inkscape, Adobe Illustrator, или онлайн редактор)
2. Найдите координаты каждого кабинета:
   - Для прямоугольных кабинетов: левый верхний угол (x1, y1) и правый нижний угол (x2, y2)
   - Для многоугольных кабинетов: все точки многоугольника

3. Обновите координаты в `MainActivity.kt`:
   ```kotlin
   areas.add(SvgMapHelper.createRectArea("room_201", x1, y1, x2, y2))
   ```

### Примеры:

**Прямоугольный кабинет:**
```kotlin
areas.add(SvgMapHelper.createRectArea("room_201", 100f, 150f, 250f, 300f))
```

**Многоугольный кабинет:**
```kotlin
areas.add(SvgMapHelper.createPolygonArea("room_204", floatArrayOf(
    100f, 100f,  // точка 1 (x, y)
    200f, 100f,  // точка 2
    200f, 200f,  // точка 3
    150f, 250f,  // точка 4
    100f, 200f   // точка 5
)))
```

**Круглый кабинет:**
```kotlin
areas.add(SvgMapHelper.createCircleArea("room_205", centerX, centerY, radius))
```

## Важно:

- Координаты в SVG обычно начинаются с (0, 0) в левом верхнем углу
- Убедитесь, что `roomId` в областях соответствует ID кабинетов в `RoomRepository`
- После изменения координат пересоберите и запустите приложение для проверки
